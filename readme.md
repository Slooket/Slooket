# Booket

Booket is a Blooket inspired game with many many additional features when compared to Blooket, we're also always looking for new exciting features to add to the game so feel free to suggest them over at our Discord! Here are some of the features we have so far:

Custom Packs
Custom Blooks
Trading
Multiplayer
Leaderboards
Chat
Friends

## Selfhost
